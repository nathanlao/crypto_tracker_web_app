## How to Run

1. Navigate to client app:
   ```bash
   cd /crypto_tracker_web_app
   ```

2. Install Dependencies:

   ```bash
   npm install
   ```
3. Run the application
   ```
   # To run client only
      npm run dev
   ```
4. Access the Client:
   
   Client app is live at <http://localhost:3000/>
